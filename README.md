# 🔗 UrlShortenerApp

Легковесный бэкенд-сервис для сокращения длинных URL-адресов и автоматического перенаправления (редиректа) пользователей. Проект реализован на архитектуре Minimal APIs.

## 🛠 Технологический стек

* **Платформа:** .NET 9 (ASP.NET Core Minimal APIs)
* **ORM:** Entity Framework Core
* **База данных:** SQLite (файловое хранилище `urls.db`)

## ⚙️ Функционал (Эндпоинты)

Сервис предоставляет два HTTP-метода:

1. **`POST /shorten`** — принимает оригинальный длинный URL в формате JSON, генерирует уникальный 6-значный хэш-код и сохраняет связку в базу данных. Возвращает сгенерированную короткую ссылку.
2. **`GET /r/{code}`** — принимает короткий код, выполняет поиск в локальной базе данных SQLite и осуществляет HTTP-редирект (302 Found) на целевой веб-сайт.

## 🚀 Запуск проекта

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в папку проекта и запустите приложение:
   ```bash
   dotnet run
   ```
   *При первом запуске база данных `urls.db` будет создана автоматически.*
